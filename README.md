## 🚀 Features

- **JavaScript function exercises**
  - Practice with simple and intermediate functions
  - Input-based logic and return values
  - Console-driven output

- **String manipulation**
  - Remove characters from strings
  - Reverse text
  - Generate acronyms
  - Check palindromes
  - Work with anagrams

- **Array operations**
  - Sum array values
  - Check conditions inside arrays
  - Split arrays into sub-arrays
  - Filter logical results through iteration

- **Number and condition exercises**
  - Check ranges
  - Reverse numbers
  - Classify angles
  - Work with arithmetic and boolean conditions

- **Pattern printing**
  - Stair pattern with `#`
  - Pyramid-related exercise
  - Console-based visual output

- **Educational Context**
  - Built as a JavaScript exercise collection to improve logic, iteration and problem-solving skills
